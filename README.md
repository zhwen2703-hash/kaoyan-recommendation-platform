# 全国考研招生单位实时数据平台

项目目录：`E:\考研推荐平台-真实数据版`

## 数据真实性原则

- 招生单位目录来自中国研究生招生信息网院校库：https://yz.chsi.com.cn/sch/
- 当前官方快照包含 939 个硕士招生单位。
- 每条记录保留官方来源链接和同步时间。
- 招生人数、报考人数、复试线等未从官方公告解析成功时显示“未公开”，不生成估算值。

## 实时更新机制

- `npm run sync:data`：立即重新抓取研招网全部分页并原子更新本地快照。
- `POST /api/sync`：网站内点击“立即同步最新数据”触发强制刷新。
- `/api/schools`：读取快照；快照超过 6 小时会自动重新同步。
- API 返回 `Cache-Control: no-store`，前端不会使用旧接口缓存。
- 408 快照超过 1 小时后，查询接口会自动启动后台全量核验；同步期间继续提供上一版可用数据，完成后原子切换。
- 408 页面每 5 分钟自动重新请求接口，并显示官方数据同步时间及后台更新状态。

## 专业与考试科目

- 网站统一通过“按考试科目选学校和专业”模块查询408院校、专业、院系、培养方式及四门初试科目。
- `npm run sync:408`：逐一核验 939 个招生单位，生成严格统考 408 全量快照。
- 408 清单只收录第四门初试科目代码精确等于 `408` 的记录，不把自命题“计算机综合”等科目算作 408。
- 每条专业记录独立标明“学硕 / 专硕”；优先使用官方学位类型，官方字段缺失时依据专业代码规则补全。
- 408 结果关联招生单位目录，显示研究生院、自划线和双一流标签。
- 支持筛选双一流/非双一流；“本科背景友好度”仅作为带来源的网络口碑，分为相对友好、存在争议和信息不足，不作为学校官方事实。
- 院校层级按教育部历史工程名单分别标注985、211和双一流；985高校同时保留211标签，普通院校表示三类标签均不属于。
- 分别显示 2026、2027 拟招生人数；两年人数都已公开时自动计算扩招、缩招或持平，2027 未发布时明确显示“未公布”。
- 支持按英语二（204）、数学二（302）精确筛选，可组合查询“408 + 英二 + 数二”。
- 复录比按专业的“复试人数 ÷ 拟录取人数”计算；只有官方名单可核验时才展示，缺失时显示“未采集”。
- 非研招网复录比单独标为“网络参考”，显示来源、计算人数和置信度；官方名单核算优先，权威机构汇总次之，不把普通帖子直接当事实。
- 408专业表显示带年份的专业复试线参考，并支持最高分筛选；学校线、国家线和专业线不混用，网络参考单独标注。
- 当前快照覆盖 389 个招生单位、10 类计算机相关专业代码，共 1569 条学校专业记录。

## 近年数据

- 每个招生单位固定展示 2026、2025、2024、2023 四个年度。
- 指标包括复试线、招生人数和复录比。
- 2026 年数据始终保留在首行；未公布时显示“未公开”。
- 更新招生单位目录时会合并并保留已经采集的历史指标，不会删除旧年度或今年数据。

## 本地运行

```powershell
cd 'E:\考研推荐平台-真实数据版'
npm install
npm run sync:data
npm run sync:408
npm run dev
```

打开 http://localhost:3000

公网免费部署参见 `DEPLOY.md`。推荐使用 GitHub Actions 每日同步数据，并由 Vercel 自动部署最新快照。

## 重要边界

“全部学校”按研招网硕士招生单位口径，包括高校、科研院所和其他招生单位。专业与初试科目直接查询研招网当年专业目录。复试线、历史招生人数和复录比需要针对各单位官网建立年度解析适配器；项目当前不会把缺失数据伪造成真实值。
